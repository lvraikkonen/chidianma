# 中午吃点啥领域语言

本文件定义团队推荐与轻量身份领域中的统一用语。它只说明术语含义，不替代产品规格或实施计划。

## 身份与访问

**轻量身份（Lightweight Identity）**：
由产品生成、可被多个已连接客户端共同使用的参与者标识；姓名不唯一，也不证明现实中的个人身份。
_避免使用_：账号、用户账号、已验证身份

**身份连接码（Identity Link Code）**：
由一个仍持有有效轻量身份的客户端临时生成、供另一客户端一次性连接到同一身份的短期代码。
_避免使用_：恢复码、登录密码、小组邀请码

**小组邀请码（Group Invite Code）**：
允许一个轻量身份申请加入特定小组的可轮换代码；它不创建或恢复轻量身份。
_避免使用_：身份连接码、恢复码

**本机断开（Disconnect This Device）**：
仅移除当前客户端保存的身份与小组访问能力，不影响已连接的其他客户端。
_避免使用_：注销账号、删除身份、撤销所有会话

**重置所有连接（Reset All Connections）**：
撤销某一轻量身份在全部客户端上的既有访问能力，并让发起重置的客户端取得新的访问能力。
_避免使用_：本机断开、单设备撤销

**身份参考号（Identity Reference）**：
用于支持沟通和核对轻量身份的稳定标识；它不是姓名、密码或现实身份凭证。
_避免使用_：账号、用户名、登录名

## 小组与成员

**小组成员资格（Group Membership）**：
轻量身份与某一小组之间的关系，包含该小组内的角色和当前状态。
_避免使用_：账号权限、全局角色

**移除成员（Remove Member）**：
终止一项小组成员资格；不会删除对应轻量身份，也不会影响其在其他小组中的成员资格。
_避免使用_：删除用户、封禁身份

**Operator 恢复（Operator Recovery）**：
当小组唯一管理员失去全部有效访问能力时，由运营人员核验已知同事关系后完成的管理员成员资格替换。
_避免使用_：找回账号、恢复码

## 隐私

**匿名化身份（Anonymize Identity）**：
撤销轻量身份的活动成员资格并移除其可识别资料，同时保留去标识化的历史归因和统计关系。
_避免使用_：删除全部历史、停用账号
